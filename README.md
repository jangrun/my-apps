# 🚀 My App Portfolio & Legal Docs

이 저장소는 제가 개발하고 배포하는 안드로이드 앱의 **소개 페이지**와 **개인정보처리방침(Privacy Policy)**을 통합 관리하는 공간입니다.

## 📂 프로젝트 구조 (Project Structure)

모든 앱은 `apps/` 디렉토리 내 독립된 폴더로 관리하여, 향후 확장이 용이하도록 설계되었습니다.

```text
.
├── README.md               # 저장소 관리 가이드
├── index.html              # 전체 앱 포트폴리오 메인 (대시보드)
└── apps/
    └── time-dial/          # [App] Time Dial 프로젝트
        ├── index.html      # 앱 상세 소개 및 기능 가이드
        └── privacy.html    # Google Play Store 등록용 개인정보처리방침
